# 雪球关注列表CSV格式参考

## 标准格式

```csv
代码,名称,行业,备注
600519,贵州茅台,白酒,核心持仓
000858,五粮液,白酒,观察
```

## 字段说明

| 字段名 | 别名 | 类型 | 说明 |
|--------|------|------|------|
| 代码 | code | string | 股票代码 |
| 名称 | name | string | 股票名称 |
| 行业 | industry | string | 所属行业 |
| 备注 | note | string | 用户备注 |

## 数据流向Wiki

关注列表数据可流向以下Wiki维度：
- 投资/能力圈/关注行业
- 投资/风格画像
