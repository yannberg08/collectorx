# 东方财富交割单CSV格式参考

## 标准格式

```csv
成交日期,成交时间,证券代码,证券名称,买卖方向,成交价格,成交数量,成交金额,手续费,印花税,过户费
2024-01-15 09:30:00,600519,贵州茅台,买入,1800.00,100,180000.00,54.00,0.00,1.80
```

## 字段说明

| 字段名 | 别名 | 类型 | 说明 |
|--------|------|------|------|
| 成交日期 | 日期/date | string | 交易日期 |
| 成交时间 | 时间/time | string | 交易时间 |
| 证券代码 | 代码/code/stock_code | string | 股票代码 |
| 证券名称 | 名称/name/stock_name | string | 股票名称 |
| 买卖方向 | 方向/direction | string | 买入/卖出 |
| 成交价格 | 价格/price | float | 成交价格 |
| 成交数量 | 数量/quantity | int | 成交数量（股） |
| 成交金额 | 金额/amount | float | 成交金额（元） |
| 手续费 | 佣金/fee | float | 手续费（元） |
| 印花税 | tax | float | 印花税（元） |

## 导出方法

1. 打开东方财富APP
2. 进入"交易" → "查询" → "交割单"
3. 选择日期范围
4. 点击"导出"按钮
5. 选择CSV格式保存

## 数据流向Wiki

交割单数据可流向以下Wiki维度：
- 外在/履历/交易记录
- 投资/交易历史
- 投资/持仓变化
