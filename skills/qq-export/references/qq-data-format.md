# QQ数据格式参考

## 数据目录位置

### macOS
```
~/Library/Containers/com.tencent.qq/Data/Library/Application Support/QQ/<QQ号>/Msg/
```

### Windows
```
C:\Users\<用户>\Documents\Tencent Files\<QQ号>\Msg\
```

### Linux
```
~/.local/share/QQ/<QQ号>/Msg/
```

## 数据流向Wiki

QQ聊天数据可流向以下Wiki维度：
- 外在/关系/联系人
- 外在/关系/对话记录
- 内在/知识体系/信息源
